Sobre meu aprendizado

Desde que comecei o curso da Trybe encontro novos desafios diariamente, seja adiquirindo novos conhecimentos e habilidades ou interagindo com outros estudantes. Através dos fundamentos trabalhados, pude conhecer o sistema Unix e o Shell, além de aprender a utilizar o Git e a plataforma GitHub para melhor desenvolver as atitividades propostas. Também aprendi as estruturas básicas de HTML e CSS, podendo assim criar este projeto e minhas primeiras páginas web.

Nunca sozinho!

Aqui na Trybe percebi que o melhor para chegar longe na minha carreira é caminhar ao lado dos demais estudantes, formando amizades duradouras e compartilhando conhecimento. Quando conquistamos juntos, conquistamos mais!
