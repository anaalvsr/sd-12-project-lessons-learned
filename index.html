Sobre o meu aprendizado

Observando meu aprendizado nesse período, desde os dias de preparação até hoje o dia do projeto, é notável que houve uma grande evolução de quando abri pela primeira vez o terminal. Imaginei que por mais que o curso tivesse um ano de duração eu ainda iria demorar bastante tempo para conseguir montar uma página na web. Porém houve uma quebra de expectativa quando em três dias acompanhando as aulas, eu vi o progresso: tinha montando minha uma página HTML e com toda a estilização que o CSS pode oferecer.

Quando estavamos na aula 3.3 do bloco de HTML e CSS, ver a evolução de uma página com texto e algumas imagens para o que se tornou, pra mim foi mágico, ver a evolução de uma sequencia de texto e imagens para uma página estilizada praticamete identica a original foi incrivel.
