Experiência Intensa

Não existe sentença melhor para resumir como é estudar na Trybe. Em apenas 3 semanas desde o início das aulas, foi tanto conhecimento adquirido que já parece que se passaram meses. A menos de um mês atrás, se me perguntassem o que é UNIX, o que é Git, commits, branchs, eu não entenderia. Talvez até afirmasse que HTML é uma linguagem de programação (e ainda passaria vergonha). Ainda há um longo caminho a ser percorrido para me tornar um bom desenvolvedor, porém cada dia que passa a certeza de estar no caminho certo cada vez maior e a realização é imensa!
